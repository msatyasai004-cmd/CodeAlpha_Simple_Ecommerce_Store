# CodeAlpha_Simple_Ecommerce_Store
A basic full-stack e-commerce platform built as part of the CodeAlpha Internship task.

## Features
- User Registration and Login
- Product Showcase & Details View
- Dynamic Shopping Cart
- Order Checkout Processing

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js, Express.js
- Authentication: Bcrypt.js, Express-Session

## How to Run
1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Start the server using `node server.js`.
4. Open `http://localhost:3000` in your browser.
